Framework

O framework é um conjunto de ferramentas,bibliotecas e convenções que facilitam o desenvolvimento de software,sites,aplicativos e outras soluções.

Front End

Alguns framework:React,Angular,Vue.js entre outros.Esses framework citados são alguns de vários que os Javascript possui.

Back End

Já no Back end temos: Django,Flask,Spring Boot,Ruby entre demasi outros framewok.Os frameworks citados não são de uma linguagens de progrmação especifica,mas são os framewoks mais utilizados no mercado de trabalho

Linguagens de Progrmação

Java,Python,Javascript,C,C++,#.Essas são as lingugens de programação mais reconhecidas e mais utilizadas

Orientação a Objeto

É um modelo de software baseado na abstração digital do mundo real,através da composição e interação entre diversas unidades chamadas de objetos e as classes.
